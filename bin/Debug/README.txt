# Bankbot - Ready to Run

This folder contains everything you need to run the Bankbot!

## Quick Setup:

1. Edit launcher-config.json with your login details
2. Edit config.json - change "YourBankbotCharacterName" to your actual character name
3. Add your trusted users to the AuthorizedUsers list
4. Run: BankbotLauncher.exe

## Commands (send via private message):
- help - Show available commands
- status - Show bot status  
- trade - Start a trade session
- return - Return your saved items

Everything runs from this single folder!
